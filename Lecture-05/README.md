Skills section (6+ skills in grid layout)

Click "Toggle Theme" button → page switches theme + saves to localStorage
Reload page → theme preference is remembered
Check footer → "Last updated" shows current date (YYYY-MM-DD)
Click "Click Counter" button → counter logs in console
